# 🌐 public

**public** est un dépôt destiné à **stocker et partager tous types de fichiers publics** utilisés dans mes projets, READMEs, tutoriels ou documentations. 🚀📂  
Il centralise **images, GIFs, vidéos, documents, audio et autres ressources** afin de les rendre facilement accessibles et réutilisables dans différents projets.

---

## 🎯 Objectifs du dépôt
- Fournir un **répertoire centralisé** pour tous types de fichiers :  
  - Images : `.png`, `.jpg`, `.jpeg`, `.svg`, `.webp`  
  - GIFs animés : `.gif`  
  - Vidéos : `.mp4`, `.mov`, `.avi`, `.webm`  
  - Documents : `.pdf`, `.docx`, `.xlsx`, `.txt`  
  - Audio : `.mp3`, `.wav`  
  - Archives : `.zip`, `.rar`, `.tar.gz`  
- Faciliter l’intégration de ressources dans les **READMEs, tutoriels ou présentations**.  
- Partager des fichiers libres ou personnels pour les projets open-source.  
- Maintenir une **organisation simple et efficace** pour retrouver rapidement chaque type de fichier.  
- Offrir un **espace public de référence** pour tous mes projets numériques.
